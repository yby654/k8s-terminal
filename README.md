# web-k8s-exec

Use WebSocket implements Docker exec via Kubernetes API.

## Getting and Start

```
git clone https://github.com/samejack/web-k8s-exec.git
cd web-k8s-exec
npm install
nodejs index.js
```
You can change configuration in index.js

## Use browser goto http://127.0.0.1:8080
